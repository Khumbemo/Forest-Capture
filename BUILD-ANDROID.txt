Forest Capture — build a debug APK (Capacitor)
==============================================

Prerequisites: Node.js, Android Studio (SDK + a device or emulator image).

1) Open terminal in this folder (Forest Logger App).

2) npm install

3) npm run cap:sync
   (copies web files to www/ and syncs into android/)

4) npx cap open android

5) In Android Studio: Build > Build Bundle(s) / APK(s) > Build APK(s)

Debug APK output (typical path):

   android\app\build\outputs\apk\debug\app-debug.apk

Transfer that file to your phone and open it to install (enable "Install unknown apps" for your file app if prompted).

First-time setup: if android\ does not exist yet, run: npx cap add android
(then npm run cap:sync again).
